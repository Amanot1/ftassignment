🌟 Luminaries - মুসলিম বিজ্ঞানীদের প্রদর্শনীএকটি আধুনিক, প্রিমিয়াম এবং ইন্টারঅ্যাক্টিভ ওয়েবসাইট যা মুসলিম স্বর্ণযুগের বিজ্ঞানীদের অমর অবদানকে তুলে ধরে। এই প্রজেক্টের সবচেয়ে বড় বৈশিষ্ট্য হলো— এটি সম্পূর্ণরূপে JavaScript (JS) ছাড়াই তৈরি করা হয়েছে।🚀 মূল বৈশিষ্ট্য (Key Features)Zero JavaScript Logic: কোনো স্ক্রিপ্ট ছাড়াই মেনু টগল এবং নেভিগেশনাল ইন্টারঅ্যাকশন কাজ করে।Checkbox Hack Implementation: মোবাইল মেনু শো/হাইড করার জন্য বুদ্ধিদীপ্তভাবে CSS Checkbox পদ্ধতি ব্যবহার করা হয়েছে।Pure CSS Animations: শুধুমাত্র keyframes ব্যবহার করে ফ্লোটিং ব্যাকগ্রাউন্ড এবং মরফিং ইফেক্ট তৈরি করা হয়েছে।Glassmorphism UI: আধুনিক ফ্রস্টেড গ্লাস ইফেক্ট এবং প্রিমিয়াম টাইপোগ্রাফির সমন্বয়।Smooth Scroll Navigation: HTML অ্যাঙ্কর লিঙ্কের মাধ্যমে এক সেকশন থেকে অন্য সেকশনে স্মুথ ট্রানজিশন।🛠️ ব্যবহৃত প্রযুক্তি (Technologies Used)প্রযুক্তিব্যবহারের উদ্দেশ্যHTML5সাইটের মূল কাঠামো এবং লজিক্যাল আর্কিটেকচারের জন্য।Tailwind CSSদ্রুত এবং রেসপন্সিভ ইউটিলিটি-বেসড স্টাইলিংয়ের জন্য।Custom CSSঅ্যাডভান্সড অ্যানিমেশন, গ্লাসমরফিজম এবং নো-জেএস লজিকের জন্য।🖼️ ইমেজ হ্যান্ডলিং স্ট্র্যাটেজি (Image Handling)গিটহাব বা লাইভ হোস্টিংয়ে ইমেজ পাথ ডিটেক্ট করার সমস্যা এড়াতে এই প্রজেক্টে বিশেষ ব্যবস্থা নেওয়া হয়েছে:Direct Image Links: সরাসরি রুট ডিরেক্টরি ব্যবহার করা হয়েছে যাতে ব্রাউজার সহজেই ফাইল খুঁজে পায়।External Hosting: ইমেজগুলো যাতে দ্রুত এবং নির্ভুলভাবে লোড হয়, সেজন্য সরাসরি অনলাইন ইমেজ ইউআরএল (URL) ব্যবহার করা হয়েছে।🧠 টেকনিক্যাল লজিক: জাভাস্ক্রিপ্ট ছাড়া কীভাবে সম্ভব?জাভাস্ক্রিপ্ট ছাড়াই ইন্টারঅ্যাক্টিভিটি যোগ করা এই প্রজেক্টের প্রধান চ্যালেঞ্জ ছিল। এটি যেভাবে কাজ করে:১. মেনু টগল (The Checkbox Hack)আমরা একটি লুকানো <input type="checkbox"> ব্যবহার করেছি। যখনই ইউজার মেনু আইকনে (যা একটি <label>) ক্লিক করে, চেকবক্সটি checked হয়ে যায়। সিএসএস-এর ~ (Sibling Selector) অপারেটর ব্যবহার করে আমরা মেনুটিকে কন্ট্রোল করেছি:CSS/* মোবাইল মেনু লজিক */
#mobile-menu-toggle:checked ~ .mobile-menu {
    transform: translateX(0);
    display: block;
}
