🌟 Luminaries - Muslim Scientists Exhibition
একটি আধুনিক, প্রিমিয়াম এবং ইন্টারঅ্যাক্টিভ ওয়েবসাইট যা মুসলিম স্বর্ণযুগের বিজ্ঞানীদের অবদানকে তুলে ধরে। এই প্রজেক্টের সবচেয়ে বড় চ্যালেঞ্জ এবং বৈশিষ্ট্য হলো— এটি সম্পূর্ণরূপে JavaScript (JS) ছাড়া তৈরি করা হয়েছে।

🚀 মূল বৈশিষ্ট্য (Key Features)
Zero JavaScript Logic: কোনো স্ক্রিপ্ট ছাড়াই মেনু টগল এবং নেভিগেশন কাজ করে।
Checkbox Hack Implementation: মোবাইল মেনু শো/হাইড করার জন্য বুদ্ধিদীপ্তভাবে CSS Checkbox পদ্ধতি ব্যবহার করা হয়েছে।
Pure CSS Animations: Keyframes ব্যবহার করে ফ্লোটিং ব্যাকগ্রাউন্ড এবং মরফিং ইফেক্ট তৈরি করা হয়েছে।
Glassmorphism UI: আধুনিক ফ্রস্টেড গ্লাস ইফেক্ট এবং প্রিমিয়াম টাইপোগ্রাফি।
Smooth Scroll Navigation: HTML অ্যাঙ্কর লিঙ্কের মাধ্যমে এক সেকশন থেকে অন্য সেকশনে স্মুথ ট্রানজিশন।
🛠️ প্রযুক্তি (Technologies Used)
HTML5: কাঠামোগত ডিজাইনের জন্য।
Tailwind CSS: দ্রুত এবং রেসপন্সিভ স্টাইলিংয়ের জন্য।
Custom CSS: অ্যাডভান্সড অ্যানিমেশন এবং গ্লাসমরফিজম ইফেক্টের জন্য।
🖼️ ইমেজ হ্যান্ডলিং (Image Handling Strategy)
গিটহাব বা লাইভ হোস্টিংয়ে অনেক সময় ফোল্ডারের ভেতর ইমেজ ডিটেক্ট করতে সমস্যা হয়। এই সমস্যা সমাধানের জন্য এই প্রজেক্টে:

Direct Image Links: আমি সরাসরি ইমেজ লিঙ্ক বা রুট ডিরেক্টরি ব্যবহার করেছি যাতে ব্রাউজার সহজেই ইমেজ খুঁজে পায়।
External Hosting: ইমেজগুলো যাতে লোড হতে সমস্যা না হয়, সেজন্য কোডে সরাসরি অনলাইন ইমেজ ইউআরএল (URL) ব্যবহার করা হয়েছে।
🧠 টেকনিক্যাল লজিক (How it works without JS?)
১. কেন বাটন বা মেনুবার ক্লিকেবল? (The Menu Toggle)
আমরা একটি লুকানো <input type="checkbox"> ব্যবহার করেছি। যখনই ইউজার মেনু আইকনে (যা একটি <label>) ক্লিক করে, চেকবক্সটি checked হয়ে যায়। CSS-এর ~ অপারেটর ব্যবহার করে আমরা মেনুটিকে নিয়ন্ত্রণ করেছি:

#mobile-menu-toggle:checked ~ .mobile-menu {
    transform: translateX(0);
    display: block;
}
২. স্মুথ নেভিগেশন (#home, #timeline)
সেকশন পরিবর্তনের জন্য জাভাস্ক্রিপ্টের onClick ইভেন্টের বদলে HTML-এর বিল্ট-ইন Anchor Tags ব্যবহার করা হয়েছে। আর স্ক্রলিং যাতে দেখতে সুন্দর হয়, সেজন্য সিএসএস-এ দেওয়া হয়েছে:

CSS
html {
    scroll-behavior: smooth;
}
৩. ফ্লোটিং ব্যাকগ্রাউন্ড
কোনো এক্সটার্নাল লাইব্রেরি ছাড়াই শুধুমাত্র CSS @keyframes ব্যবহার করে ব্যাকগ্রাউন্ডের এলিমেন্টগুলোকে একটি লুপে অ্যানিমেট করা হয়েছে, যা ওয়েবসাইটকে প্রাণবন্ত করে তোলে।
