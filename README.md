# 🌟 Luminaries - Muslim Scientists Exhibition

> **একটি আধুনিক, প্রিমিয়াম এবং ইন্টারঅ্যাক্টিভ ওয়েবসাইট যা মুসলিম স্বর্ণযুগের বিজ্ঞানীদের অবদানকে তুলে ধরে।**

---

### 🚀 মূল বৈশিষ্ট্য (Key Features)

* **Zero JavaScript Logic:** কোনো স্ক্রিপ্ট ছাড়াই মেনু টগল এবং নেভিগেশন কাজ করে।
* **Checkbox Hack Implementation:** মোবাইল মেনু শো/হাইড করার জন্য বুদ্ধিদীপ্তভাবে CSS Checkbox পদ্ধতি ব্যবহার করা হয়েছে।
* **Pure CSS Animations:** `Keyframes` ব্যবহার করে ফ্লোটিং ব্যাকগ্রাউন্ড এবং মরফিং ইফেক্ট তৈরি করা হয়েছে।
* **Glassmorphism UI:** আধুনিক ফ্রস্টেড গ্লাস ইফেক্ট এবং প্রিমিয়াম টাইপোগ্রাফি।
* **Smooth Scroll Navigation:** HTML অ্যাঙ্কর লিঙ্কের মাধ্যমে এক সেকশন থেকে অন্য সেকশনে স্মুথ ট্রানজিশন।

### 🖼️ ইমেজ হ্যান্ডলিং (Image Handling Strategy)

গিটহাব বা লাইভ হোস্টিংয়ে ইমেজ ডিটেক্ট করার সমস্যা সমাধানের জন্য এই প্রজেক্টে:
1.  **Direct Image Links:** সরাসরি ইমেজ লিঙ্ক বা রুট ডিরেক্টরি ব্যবহার করা হয়েছে।
2.  **External Hosting:** দ্রুত লোডিং নিশ্চিত করতে অনলাইন ইমেজ ইউআরএল (URL) ব্যবহার করা হয়েছে।

### 🧠 টেকনিক্যাল লজিক (How it works without JS?)

#### ১. কেন বাটন বা মেনুবার ক্লিকেবল? (The Menu Toggle)
আমরা একটি লুকানো `<input type="checkbox">` ব্যবহার করেছি। যখনই ইউজার মেনু আইকনে ক্লিক করে, চেকবক্সটি `checked` হয়ে যায়। CSS-এর `~` অপারেটর ব্যবহার করে আমরা মেনুটিকে নিয়ন্ত্রণ করেছি:

```css
#mobile-menu-toggle:checked ~ .mobile-menu {
    transform: translateX(0);
    display: block;
}
২. স্মুথ নেভিগেশন (#home, #timeline)
সেকশন পরিবর্তনের জন্য জাভাস্ক্রিপ্টের বদলে HTML-এর বিল্ট-ইন Anchor Tags ব্যবহার করা হয়েছে। স্ক্রলিং স্মুথ করতে সিএসএস-এ দেওয়া হয়েছে:

CSS
html {
    scroll-behavior: smooth;
}
৩. ফ্লোটিং ব্যাকগ্রাউন্ড
কোনো এক্সটার্নাল লাইব্রেরি ছাড়াই শুধুমাত্র CSS @keyframes ব্যবহার করে ব্যাকগ্রাউন্ডের এলিমেন্টগুলোকে একটি লুপে অ্যানিমেট করা হয়েছে।
